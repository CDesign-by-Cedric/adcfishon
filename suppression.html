Suppression de compte - AdCFishon

Cette page explique comment supprimer votre compte et vos données associées sur l'application AdCFishon, développée par CDesign-by-Cedric.

Comment supprimer votre compte

1. Ouvrez l'application AdCFishon
2. Accédez au menu Paramètres
3. Cliquez sur Supprimer mes données puis suivez les instructions.

Données supprimées

Lors de la suppression du compte, toutes les données suivantes sont supprimées de manière définitive :
- Adresse e-mail
- Données de captures (poissons, météo, localisation...)
- Données de connexion et de sécurité (mot de passe, question secrète)

Données conservées

Aucune donnée personnelle n'est conservée après la suppression du compte. Aucune donnée n'est stockée sur des serveurs externes. Toutes les données sont locales.

Durée de conservation

Les données sont supprimées immédiatement après validation de la suppression depuis l'application.

Pour toute question, contactez-nous via GitHub ou notre adresse email associée.
